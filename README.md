# Analisis Regresi Linear dan Support Vector Classification untuk Estimasi Jumlah dan Klasifikasi Musim Kunjungan Wisatawan Nusantara di Kabupaten/Kota Provinsi Sulawesi Tengah

## Deskripsi
Proyek ini bertujuan untuk menganalisis dan memprediksi jumlah kunjungan wisatawan nusantara di Kabupaten/Kota Provinsi Sulawesi Tengah menggunakan metode Linear Regression serta mengklasifikasikan musim kunjungan wisatawan menggunakan Support Vector Classification (SVC).

## Metode
- Linear Regression
- Support Vector Classification (SVC)

## Dataset
Dataset jumlah wisatawan nusantara Kabupaten/Kota Provinsi Sulawesi Tengah periode 2024–2026.

## Output
- Grafik analisis wisatawan
- Hasil validasi model
- Hasil estimasi jumlah wisatawan
- Klasifikasi musim kunjungan wisatawan

## Cara Menjalankan

1. Buka terminal.
2. Masuk ke folder proyek.
3. Jalankan:

python analisis_wisatawan.py
